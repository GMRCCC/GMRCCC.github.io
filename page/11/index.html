上一节我们提到，我们想要找到合适的w和b，使得J(w,b)最小

为了方便图像表示，我们将w看着是一个实数(实际是一个矩阵)，则J(w,b)是在水平轴w和b上的曲面，曲面的高度表示了J(w,b)在某一点的值。

可以看到，成本函数J是一个凸函数，这也是使用这种损失函数的重要原因。

为了找到更好的参数值，我们要做的是用某初始值初始化w和b，对逻辑回归来说， 几乎任何的初始化方法都是有效的 ，通常用0来进行初始化，由于函数是凸函数，无论采用怎样的初始值，优化后的值都是接近的。

梯度下降法所做的就是，从初始点开始，朝最陡的下坡方向走一步，这就是梯度下降的一次迭代。

2.6 使用计算图求导

发表于 2023-10-26 更新于 2023-10-30 分类于 吴恩达深度学习1-神经网络与深度学习

这是上一节的流程图，假设你要计算J对v的导数，该如何计算？

我们让v变化一点点，比如从11变为11.001，此时J从33变为33.003，则可以很容易求出J对v的导数为3，这样我们就做完了一步反向传播。

那我们该如何计算J对a的导数呢？

我们让a从5变为5.001，则v从11变为11.001，则v对a的导数就是1，J从33变为33.003，则J对a的导数就是3

一种理解方法是用链式求导法则，我们先算J对v的求导，然后再算v对a的求导，这样J对a的求导就为两者的乘积，这样就完成了两步反向传播。

现在介绍一个新的符号规定，当你编程实现反向传播时，通用有一个最终输出值是你要关心的，很多计算尝试计算输出变量的导数，比如我们算FinalOutputval对var的导数，通常来将需要很长的变量去表示，在代码中，我们直接简化为dvar即可，比如算J对v的导数，用dv表示即可。

这都是很简单的求导计算，最重要的一点是要知道当计算所有导数时，最有效率的办法是从右到左计算，这样我们在前面计算的导数结果就可以在后面用到。

这就是一个计算流程图，就是正向即从左到右计算成本函数J，然后反向从右到左计算导数。

2.9 向量化

发表于 2023-10-26 更新于 2023-10-30 分类于 吴恩达深度学习1-神经网络与深度学习

向量化对加速我们的计算过程十分重要

如上图所示，左边是非向量化的计算过程，我们必须用for循环去计算，速度很慢。右边是numpy中内置的点乘，用这种方法不仅速度快，而且代码也简洁了许多。

我们可以用jupyter做一个验证

可以看到计算结果是一样的，但用for循环的耗时是用向量化方法的300多倍。

使用向量化方法快的原因是，无论GPU还是CPU，它们都有并行化指令即SIMD指令(单指令多数据流)，numpy能充分的利用并行化去更快的计算。GPU对CPU的优势是它更擅长做SIMD的计算。

一个经验法则是，只要有其他方法，我们应该避免使用for循环来提高我们的计算速度。

3.11 随机初始化

发表于 2023-10-26 分类于 吴恩达深度学习1-神经网络与深度学习

首先我们要知道，将所有权重初始化为0是没有意义的

如图所示，假设W为一个2×2的零矩阵，这意味隐藏层的两个神经元参数完全一样，我们将同样的样本输入进去，经过梯度下降后，更新的参数也会完全一样(很符合直觉，所有的样本和权重都一样，更新后当然也一样，可以用归纳法证明)。

这意味着无论你训练多少次，隐藏层的两个神经元训练出来的函数都相同，效果等于只用了一个神经元。

所以我们需要随机初始化权重，保证多个神经元是有意义的

b可以设置为0(w要不同，因为梯度求导时看w而不看b)，W用np的random函数进行随机初始化，再×0.01来缩小，这样算出来的z也较小，这样做是因为我们用的激活函数(sigmoid、tanh)一般在0附近的斜率较大，让z值落在0附近可以保证梯度下降的速度。

当神经网络的深度很大时，我们也可以使用0.01以外的常数，这将在以后进行讨论。

先确保所有向量的维度都是对的，这样就能消除很多bug

仔细推导后可以发现维度匹配，注意反向传播时有些矩阵需要转置(比如a、W)才能匹配。单个样本的推导和逻辑回归那边非常类似，区别不过是一个推一次一个推两次，第一次是多个神经元且函数不确定罢了。

3.2 神经网络表示

发表于 2023-10-26 更新于 2023-10-30 分类于 吴恩达深度学习1-神经网络与深度学习

我们先来看一下只有一个隐藏层的神经网络

先来给神经网络各层命名，从左到右分别为输入层、隐藏层和输出层。

隐藏层的含义是：在训练集中，这些中间节点的真正数值我们是不知道的，即你不能在训练集中看到它们，你只能看到输入值和输出值。

我们常用向量X来表示输入特征，但其实输入特征还有另一种表达方式，即a^[0]，a意味着activate，意味着网络中不同层的值会传递给后面的层。

将a^[0]输入隐藏层后，隐藏层会输出a^[1]，由于图片中的隐藏层中有4个神经元，因此a^[1]是一个四维的列向量，用a_i^[1]表示。

最后的输出层输出的值称为a^[2],这个值即为yhat的值。

使用这种记号我们能够清楚的知道输出值来自哪个层。

图片中的神经网络称为双层神经网络，在计算层数时不计入输入层，因此隐藏层为第一层，输出层是第二层，而输出层被称为第0层，不看作标准的层。

隐藏层和输出层是带有参数的，同样用上标表示其来自哪一层，由于不同层的神经单元数不同，因此对应参数的维度也会不同。
