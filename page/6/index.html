1.4 Padding

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

卷积有两个缺点，一个是会导致图像不断地缩小(有的时候你不希望图像缩小，比如有100层的卷积神经网络，如果每过一层图像都会缩小的话，最后只能得到一张非常小的图片)，另一个是图片边缘的像素使用率过低，导致大部分图像边缘的信息都丢失了。

所以在卷积之前，我们可以先填充(paddiing)图像。

padding的大小意味着填充多少层，padding=1时，填充一层，填充的值习惯上为0

此时你用3×3的卷积核去卷积，卷积后的图像就和卷积前的图像一样大了，这样检测不到图像边缘信息的问题也得到了一定程度的解决。

填充的选择通常有两种，Valid卷积和Same卷积，一个意味着不填充，一个意味着填充后卷积的图像大小与卷积前相同，此时填充的层数p=(卷积核大小f-1)/2，为了保证p为整数，f通常取奇数。

f为奇数有两个原因，一个是方便填充避免不对称的填充，另一个是方便寻找中心点，中心点对确定卷积核位置非常有用

1.5 卷积步长

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

当卷积步长为s不为1时，卷积后的图像大小为(n+2p-f)/s+1向下取整，其中n为图像大小，p为填充层数，f为卷积核大小，s为步长

在数学或者信号处理的教材上，卷积之前往往还需要一步操作，就是沿水平和垂直方向翻转，然后再进行卷积。而我们的卷积并没有这一步操作，有时候被称为相互关(cross-correlation)而不是卷积。但在深度学习惯例中，我们还是称这个操作为卷积。

当然，这并不影响我们搭建神经网络。

1.6 三维卷积

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

对一张三通道的图片，我们可以将卷积核也拓展为三维进行卷积，注意，卷积核的第三维度必须和图片的通道数相等

卷积过程和二维卷积类似，也是卷积后全部加起来得到一个值

三维卷积的好处是你可以只针对某个通道做检测，比如你想对红色通道做垂直检测，你可以将其他通道的卷积核全设为0

如果你想检测图像的多个特征(比如同时检测垂直边缘和水平边缘)，那么我们可以使用多个卷积核进行卷积，最后得到图像的通道数等于我们的卷积核数

1.7 单层卷积网络

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

和梯度传播中的类似，卷积的过程类似于乘权重的过程，然后利用广播机制我们可以对输出的图像做ReLU，最后得到输出图像，整个过程构成一个卷积层

最后输出图像的通道数等于卷积核的组数。

假设我们有10组卷积核，每个卷积核为3×3×3，那么我们会有多少个参数？

权重数为3×3×3×10=270，每个卷积核还需要1个偏差，则总参数为270+1×10=280个

使用卷积核的一个好处是，无论图片有多大，只要卷积核的大小和组数固定，那么使用的参数个数也固定。这就是卷积神经网络的一个特征，”避免过拟合”。

在卷积神经网络中有三个经典的层，卷积层、池化层和全连接层，将在之后进行说明

1.10 卷积神经网络示例

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

来看一个简单的卷积神经网络，模仿自LeNet-5

对于层数的定义，由于池化层中并没有实际要调整的参数，有些人可能会把卷积层和池化层一起看做一层，有些则是会看做两层。上图中归为了一层。

全连接的操作和神经网络中的操作一样，定义权重矩阵W和偏置b，计算从而改变特征个数。由于改变后的每个特征值都用到了改变前的每个特征值，所以叫全连接层。

卷积神经网络的趋势是图像高度(nH)和宽度(nW)不断变小，通道数(nC)不断变大。

卷积神经网络的常见操作是在一层或几层卷积层后跟一层池化层，在最后用几层全连接层，然后跟一层softmax层。

我们来看一下这个卷积神经网络的参数构成

  1. 池化层没有参数，卷积层的参数也比较少，参数数量主要集中在全连接层。
  2. 输出的特征数总体呈下降的趋势，当然不能下降得太快，否则会影响神经网络的效果。

卷积神经网络的基本模块就是卷积层、池化层和全连接层，而我们要做的是如何将这些模块合理地拼接起来，从而得到一个高效的卷积神经网络。常见的做法是阅读大量的文献，从别人的模型中获取自己对如何构造卷积神经网络的直觉，或者直接用别人的模型。

1.9 池化层

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

如图，池化核大小为2，步长为2，则相当于在原图中四个象限分别提取最大值。可以看出，左上角的9是一个特征(不管是什么特征)，而右上角没有明显的特征。

直观理解就是，池化操作的功能就是只要在任何象限内提取到某个特征，它都会保留最大值在最大池化的输出里，如果提取不到特征(如右上角)，那么它提取到的值也会很小。

池化层的真正原理并没有很清楚，但它在实际实验中的效果很好。

和卷积核不同，池化操作虽然需要确定f和s，但并不需要学习任何参数(取最大值即可)。

值得注意的是，池化层和卷积层不同，输入图像的通道数和输出图像的通道数一直是一样的(池化没有三维卷积的概念)，池化只是做取最大值操作，并不会影响通道数大小。

还有一种操作是平均池化，但使用频率比最大池化少得多

一般只有在很深的神经网络中才会被使用，比如把7×7×1000的图像平均池化为1×1×1000的图像。

池化操作最常见的是设置f=2，s=2，这样图像大小刚好降为原来的一半；f=3，s=2也比较常用。

一般而言，池化操作不进行填充，也就不需要设置padding。

2.10 数据扩充

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

计算机视觉不同于其他领域，经常会出现数据不够用的情况，因此，数据增强很有用。

常见的有镜像、随机裁剪等，由于变形、旋转等很复杂，能用但不常用(之前讲过)

另一种方法是进行颜色更改，我们可以给R、G、B三个通道的值加上或者减去一些值来改变颜色

最下面的图片模仿了更强烈的阳光或者灯光下拍出来的图片，而这种颜色变化是我们现实生活中常见的。因此使用颜色变化可以使我们的模型对现实中不同色调的图片更具有鲁棒性。

我们也可以使用PCA进行颜色增强，使图片各个通道的颜色值差不多，具体细节可以去网上看

当你的数据集很小时，你可以用自己喜欢的方式进行数据增强。

当数据集很大时，人们常用CPU中的一个和几个线程进行数据增强(可以是随机的数据增强，比如某些图片用旋转，某些图片进行颜色变换)，当数据量达到mini-bach size时(假设使用小批量梯度下降)，再送到另外的CPU的线程或者GPU进行训练

这两个过程往往可以同时进行。

类似于神经网络，数据增强的过程也需要调整许多参数，一个好的方法是从网上下载别人已经开源的数据增强实现，用别人已经训练好的参数来进行自己的数据增强。

2.11 计算机视觉现状

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

大多数机器学习问题都是介于少量数据和大量数据的范围

我们拥有的目标检测数据要少于图像识别少于语音识别。

一般来说，拥有的数据越多，人们更倾向于用更简单的算法，进行更少的手工计算；拥有的数据越少，人们更倾向于更多的手工计算，实际上在少量数据时用更多人工的方法确实在目前取得了最好的效果。

教学者认为机器学习算法知识的来源有两个

  1. 标记好的数据
  2. 手工工程

所以当你没有足够多的数据时，你可以更多的考虑进行手工工程，比如一个精心设计的特性、系统，或者一个组件。

计算机视觉领域试图去学习一个很复杂的功能，这就导致能用于计算机视觉学习的数据不够多，这也是从CV发展到现在，人们设计了如此多架构的原因。在缺乏数据的情况下，想要获得良好的表现就必须对架构进行精心设计。

随着近几年CV数据量的急剧上升，手工设计的架构也逐渐减少，但仍有很多架构被设计。

在目标检测领域，由于数据量较少，你会看到非常复杂的算法实现。

幸运的时，当我们的数据集很小时，我们可以用迁移学习来帮助我们训练。

人们表现出了极大的热情在基准的数据集中训练自己的模型(打比赛)，因为在基准数据集上做的更好意味着更容易发论文。这种热情有效促进了更多高效算法的诞生，但在实际应用中基本用不到。

几个在基准数据集上表现好的方法

  1. 集成，独立训练几个网络，并平均它们的输出。这种方法在比赛中表现很好，但在实际应用中没人用，因为训练多个神经网络意味着你需要重复训练一张图片，花费了太多计算预算和时间。
  2. multi-crop，我们前面讲过数据增强，multicrop就是对一张图片进行多次切割进行训练，比如10-crop就是取一张图片，镜像一次再复制一份成四张图片，第一张取中心裁剪，第二张取四个角落裁剪，然后再取中心裁剪、角落裁剪，共生成10张图片进行训练。或许这种方法在实际应用中也有用，但它在基准训练集上的表现要好得多。

集成的问题是如果你要同时训练几个网络，那他会大量占用你的计算机内存，multi-crop也会增加你的计算时间。

可以在研究论文中使用这些方法，但不建议在实际应用中使用。

要构建应用系统，最好的方式还是从网上拿别人的架构开始训练，并且最好能找到他们的源码，因为他们帮你实现了许多繁琐细节比如学习率衰减等，而且已经在大量数据集上训练
