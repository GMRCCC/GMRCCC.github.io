4.2 One-Shot 学习

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

人脸验证需要解决的问题是你可能只有一张图片用来训练以解决问题，因为大多数员工只会上传一张图片。

One Shot学习并不适合用卷积神经网络，一方面训练集实在太小，无法训练一个健全的神经网络，另一方面，如果有新员工加入，那么我们需要额外增加一个输出，这意味着我们需要去改变神经网络的结构重新训练，显然是不合理的

因此人脸验证一般是学习”similarity”函数，即输入两张图片，判断两张图片之间的相似度来决定是否匹配，这样就算向数据库中新加入图片，也只要做相似度匹配就可以了

4.3 Siamese 网络

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

Siamese网络用来进行两张图片的相似度匹配。

我们训练一个神经网络，将一张图片输入，输出一个128维的向量。这个操作相当于”encoding”这张图片，即将这张图片用一个128维的向量重新表示。

这样两张图片就很好比较了，我们将第二张图片同样输入这个所有参数相同的神经网络中，得到第二个128维向量。

则两张图片的差异就是这两个向量之间的差异。

神经网络相当于函数，把图片映射成了一个向量。

我们要做的就是训练一个神经网络，当输入了两张图片是同一个人时，两个向量之间的差足够小，不同人时两个向量之间的差足够大。

4.4 Triplet损失

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

我们常用三元组损失函数来评估人脸识别。

在评估时，我们需要同时看三张图片，目标(Anchor，简称A)图片、和目标图片同一个人(Positive，简称P)的图片、和目标图片不是同一个人(Negative，简称N)的图片。这也是三元组损失函数名字的由来。

A和P的距离应该比A和N的距离要小，因此如最下面的公式所示，两者之差应该小于等于0.

但这样有一个问题，就是神经网络可能会把所有输出都设置成0向量，因为这样差值=0也是满足小于等于0的条件。

因此我们加上一个间隔(margin)α，用来扩大d(A,P)和d(A,N)的差距，只有两者的差距足够大时，它们之差加上α才能小于等于0.

现在我们基于三元图片组来定义损失函数了

由于损失函数肯定大于0，因此我们让d(A,P)和d(A,N)的差与0之间取最大值。

但要用这样的三元组来训练模型意味着你必须有同一个人的多张图片(至少一部分人要)，如果一个人只有1张图片是无法组成(A,P)这样的图片对的。

因此在实际应用中，我们采用的模型都是预先训练好的，即先用有同一个人多张图片的训练集进行训练，然后再对我们自己的图片进行识别。

在构建三元组时，我们不能采用随机构造三元组的方式，因为这样d(A,N)会比d(A,P)大得多，导致α几乎起不到什么作用，反向传播的效果微乎其微。我们应该尽量挑选难区分的图片，让d(A,N)和d(A,P)近似相等，这样训练出来的模型才有最好的效果。

想了解人脸识别的更多细节，可以阅读下这篇文章

通过三元组来训练一个神经网络的过程是，从训练集中构建三元组，然后进行训练使损失函数尽可能低。

现在公司所使用的数据集可能有多达百万甚至千万张图片进行训练，可以让d(A,P)变得足够小而d(A,N)变得足够大。因此在我们自己进行训练时，可以从网上去下载已经预训练好的权重使用。

4.5 面部验证和二分类

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

比起三元组损失，我们还有很多方法来计算两张图片间的相似度，比如让两张需要比较的图片通常通过Siamese神经网络进行计算得f(x)，然后利用损失函数算损失。

损失函数比如用传统的神经网络方法算(蓝色部分)，或者用X相似度方法算(绿色部分)，都能取得不错的效果。

输入两张图片，判断最后输出的yhat是1还是0来判断是否相似。

一个提高效率的方法是将存在数据库里的图片提前计算好f(x)，这样当要识别新图片时，就不用一个个重复计算数据库里的图片和新图片进行匹配，而是直接计算新图片的f(x)和数据库中已有的f(x)进行比较即可。通过这种precompute方法可以提高系统的识别效率。

这种方法与前面的三元组方法不同，每次只需要输入成对的图片，如果是同一个人y输出1，不同的人y输出0.

这里将人脸识别问题扩展为了二分类问题，这种方法的效果也很好。

4.7 什么是深度卷积网络？

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

这篇文章，可视化理解深度神经网络，我觉得非常值得一读。

神经网络的浅层部分用来提取一些低级特征。假设第一层的每个隐藏单元都提取某个特征(如右下角的图片所示)，有些提取向右45°的线，有些提取垂直边缘特征，还有些提取橘黄色的特征等等。

这些特征能最大化隐藏单元的输出。对某个隐藏单元来讲，总有几个最明显的特征能够最大化它的输出。

随着神经网络的深度不断增加，卷积核将图片不断压缩，隐藏层能够看到的特征也越来越高级(越来越宏观)

第二层就能检测要一些圆形、很细的垂线等。

最后一层已经能够识别一些很高级的特征了。

由于房屋的价格不可能为负，因此刚开始函数值为0.

这个拟合函数就可以看做是最简单的神经网络

中间的过程可以看做一个简单的神经元，它做的事情是输入面积，完成线性运算，取不小于0的值，最后得到输出的预测价格。

在神经网络中，经常能看到这样一个函数，这个函数一开始是0，然后是一条直线

这个函数叫做ReLU函数，全称是 修正线性单元(rectified linear unit) 。

上述的神经网络是一个规模很小的单神经网络，而大一点的神经网络是多个神经元堆叠而成的，就像是搭积木一样。

神经网络的一部分神奇之处是，当你搭建完神经网络后，你只需要输入x，就能得到输出y，不管训练集有多大，所有的中间过程都可以自己完成。

这个神经网络从左到右分别为输入层、中间层和输出层，我们一般不会管中间的神经元叫什么，只关心输入样本和输出。神经网络最强大的地方是，只要你给足够的x、y训练样本，它就能计算从x到y的精准映射函数。
