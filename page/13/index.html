4.4 为什么使用深层表示

发表于 2023-10-26 分类于 吴恩达深度学习1-神经网络与深度学习

深层神经网络较早的几层能学习一些低层次的简单特征，在后几层就能把简单的特征组合起来图探测更复杂的东西

另一个理论来自于电路理论，我们可以用层数深的神经网络去计算可以让需要的隐藏单元数减少，如果我们想用浅层神经网络计算时，会发现需要的隐藏单元数呈指数增长

4.6 参数vs超参数

发表于 2023-10-26 分类于 吴恩达深度学习1-神经网络与深度学习

超参数就是控制实际参数的参数，用来控制W和b，比如学习率α，迭代次数，隐藏层数，隐藏层单元数以及激活函数的选择等

其他的超参数还有momentum，mini batch size，正则化参数等等。

超参数的选择有很多的可能性，这意味着你得尝试很多不同的值。

今天的深度学习应用领域，还是很经验性的过程，我们必须不断的调整超参数来获得较优的参数值

值得注意的是，由于电脑cpu或gpu的性能不同，调试出来的最优参数值可能 过一段时间就会变化一次 ，这意味着你需要时不时去调试你的参数。

 神经网络的更深层通常比前面的层计算更复杂的输入特征。

1.1 训练_开发_测试集

发表于 2023-10-26 分类于 吴恩达深度学习2-改善深层神经网络

在以前，我们普遍认为将训练/测试集划分成7/3的比例或者训练/验证/测试集划分成6/2/2的比例比较合理。但验证集主要用于调整超参数，测试集主要用于测试模型性能，需要的数据量不用很多。随着大数据时代到来，我们动辄需要处理百万级的数据，这时候验证/测试集的比例应该缩小，可能不到1%。

注意： 保证你的验证集和测试集来自同一分布 。

因为我们用验证集来评估不同的模型，尽可能优化性能，让验证集和测试集有同一分布的效果是非常好的。

有时候由于各种原因，你的训练集和验证/测试集可能不是来自同一分布，但只要遵循这个原则就能让算法变得更快。

最后，就算没有测试集也不要紧，测试集主要对神经网络进行无偏评估，如果你不需要无偏评估，也可以不设置测试集。这时候，人们常把验证集称为测试集(test set)。

1.11 神经网络的权重初始化

发表于 2023-10-26 分类于 吴恩达深度学习2-改善深层神经网络

为了避免出现梯度消失或者梯度爆炸的问题，我们应该慎重选择权重初始化的方法

对如tanh函数，可以选择Xavier初始化；对RELU函数，可以选择Kaiming初始化，这里不给出推导过程，只给出结论

在应用中只需知道有这么一个初始化即可，若想更进一步可以去看概率论或者论文证明。

权重初始化的目的是给权重一个固定的初始方差。我们也可以将权重方差参数化(比如乘一个乘数参数)，但大多数时候效果一般，优先级相对比较低。

1.13 梯度检验

发表于 2023-10-26 分类于 吴恩达深度学习2-改善深层神经网络

先把W[L]、b[L]矩阵转为向量，再把这些向量拼接在一起，称为θ，同样的，对dW和db也做类似的处理

现在的问题是dθ和J的梯度有什么关系

θ是一个多维的向量，我们将其中各项命名为θ1，θ2，…，然后用近似法求各个维度的梯度

最后可以得到一个dθapprox，即dθ的近似值。

将该值与精确值dθ(即dW、db组成的dθ)求差(一般是欧式距离)，如果差值很小的话，说明梯度检验没问题，可以继续往下做；如果差值过大，我们就需要考虑是不是哪里出现了bug，需要对我们的代码进行一定的修改了

1.14 关于梯度检验实现的注记

发表于 2023-10-26 分类于 吴恩达深度学习2-改善深层神经网络
  1. 不要在训练中使用梯度检验，它只用于debug，求dθapprox的过程太过漫长，所以只在debug时用
  2. 如果梯度校验失败(dθapprox和dθ相差过大)，检查dθ的每一项看一下是哪一项出了问题，θ的各项与b和w的各项都是一一对应的，如果dθapprox的某一项与dθ相差过大，我们可以由此估测要在哪一层追踪bug
  3. 如果使用正则化，请注意要包括正则项
  4. 梯度检验不能和dropout一起使用，每次迭代过程中dropout会随机消除隐层单元的不同子集，难以计算代价函数J，所以很难用梯度检验。
  5. 可以对随机初始化值进行梯度检验，训练一段时间后，再进行梯度检验(不经常做)

1.10 梯度消失和梯度爆炸

发表于 2023-10-26 分类于 吴恩达深度学习2-改善深层神经网络

假设有两个输入特征，L层，每层两个神经元，使用线性激活函数，则当W[L]矩阵的值比1大或者小时(就是只差一点点)，整个激活函数(或者梯度函数)也会呈指数级增长，导致梯度消失或梯度爆炸。如果你有很多层神经网络比如150层，这个现象会更加明显

1.2 偏差_方差

发表于 2023-10-26 分类于 吴恩达深度学习2-改善深层神经网络

如果偏差很高的话，可能会欠拟合；如果方差很高的话，可能会过拟合

方差和偏差也可以用训练集和验证集来判断，如果训练集上的错误率低但验证集上的错误率高，说明过拟合，方差高，反正说明欠拟合，偏差高。

上图的分析都假设人眼辨别的错误率接近0%，即最优误差(基本误差)接近0%。

如果最优误差非常高，比如15%，那么第二种的错误率是合理的。

上述的分析过程都假设你的基本误差很小，且训练集和验证集的数据来自相同分布。

有时也会同时存在高偏差和高方差的情况，即在局部存在过拟合和另一个局部存在欠拟合，这在数据高维时出现也是合理的。
