3.2 特征点检测

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

我们也可以用特征点(x,y)来对物体进行检测。

假设你有一个人脸，我们可以用四个坐标来输出四个眼角的位置以定位眼睛，或者用更多的特征点来定位眼睛、鼻子、嘴巴和脸部轮廓等。比如我们用64个特征点进行面部表情的识别，需要做的是将图像输入一个卷积网络，然后输出一组数，第一个数也是用来识别是否存在物体，后面的数用来表示特征点位置。

这还只是基本的面部识别模块，如果想要识别更多特征，你需要标注更多的数据点进行识别，这些数据点都是手工标注的(简直是噩梦)。

同样的，如果你想进行姿态检测，你也可以将人体的关键节点标注出来(头、肩膀、手部、膝盖等等)进行训练

用特征点检测可以做非常多有趣的事，注意的是特征点标注的特征必须在所有图片中保持一致(比如特征点1标注左下眼角，那么它在所有图片中都应该标注左下眼角)，当你有庞大的标注数据集时，实现这些也并不显得困难。

3.3 目标检测

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

使用基于滑动窗口的目标检测算法进行对象检测。

假设我们要检测车辆，先训练一个卷积网络，输入是裁剪过的让汽车占据图像大部分位置的图片，输出是是否有车

然后就可以进行滑动窗口目标检测了，我们先选定一个框，然后让这个框从左上角开始以固定速度向右下角移动，每次移动都判断框内有无汽车。第二次我们可以采用更大的框再同样检测一遍，然后第三次、第四次…。这种通过框的移动来确定车辆位置的算法就是滑动窗口检测

这种方法的问题就是计算成本太高。在CV发展早期，由于图片分类器都较为简单，分类速度较快，因此用滑动窗口进行定位的效果不错。但随着分类器的复杂度上升，滑动窗口定位速度已经太慢，而且必须用很精细、缓慢的移动才能准确定位车辆。

3.4 滑动窗口的卷积实现

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

单纯使用卷积网络进行滑动窗口的效率很慢，而在卷积层上应用这个算法则会快很多。

先来看如何用卷积操作实现全连接层

对5×5×16的图像，我们同样使用5×5×16的卷积核卷积来模拟全连接，使用400个卷积核，这样输出的维度就是1×1×400。注意这和全连接的操作完全不一样，全连接是将图像直接展平然后用经典神经网络的方法处理。

之后再使用一个1×1×400的卷积核进行全连接，这里就和全连接操作类似了，只不过一个是纵向一个是横向。

最后由1×1卷积后softmax输出。

了解了卷积实现全连接的思路，我们就可以尝试用卷积实现滑动窗口了，思路来自下面的论文

传统的滑动窗口实现是在一张图片中截取各个部分的图片，然后反复用卷积网络去计算各个部分的输出，这样会导致这些图片的公共区域被重复计算，浪费大量计算资源

而卷积神经网络则避免了公共部分的重复计算，下面的卷积过程代替了四次卷积神经网络的计算(很好理解，多看看就知道)，最后输出的2×2的方块可以替代左上、右上、左下、右下四张裁剪图片分别进行前向传播的计算

卷积过程其实就是对图片的放缩过程，比如5×5卷积就相当于按照每个区域5×5的方式等比例放缩。则max pool=2就相当于以2的步长进行窗口滑动。

用卷积来进行滑动窗口的计算，我们就不需要把原图片进行一张一张的切割分别进行计算，而是将整张图片输入直接进行计算

3.5 Bounding Box预测

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

滑动窗口的卷积检测效率很高，但不能输出最精准的边界框

YOLO(you only look at once)算法能得到更加精准的边框。

YOLO算法将一张图片划分为好几个网格，然后给每个网格贴上之前在目标定位用过的标签。如果一个物体占据了好几个网格，那么就取这个物体的中心点，中心点在哪个网格它就属于哪个网格。如果某个网格内包含物体但物体的中心点不在该网格，我们也认为这个网格不包含物体

假设我们将其划分为3×3的网格，每个网格有一个八个参数的标签，那么最后输出的数据就是3×3×8，我们可以通过卷积网络完成这件事情。

  1. 每个物体的中心点只会属于一个网格
  2. 一个网格中只会检测一个物体。为了减少一个网格中同时出现多个物体的概率，实际中我们会用更小的网格大小(比如19×19)
  3. 通过标签中的四个参数可以定位更加精确的目标框
  4. YOLO算法的效率很高，因为它也是使用单次卷积网络来输出结果，而不是不同网格重复在神经网络上跑，能够实现实时识别

YOLO锚定框的过程和目标定位中类似，我们先确定物体中心点在哪个网格，网格的左上角坐标为(0,0)，右下角坐标为(1,1)，以网格的长度来确定中心点位置和长宽大小，注意物体的长宽可以大于1，因为有些物体可能横跨多个网格。

感兴趣可以看一下YOLO的论文，论文的难度很高，一次看不懂没关系

3.7 非极大值抑制

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

非极大值抑制可以确保你对每个物体只检测一次。

在进行中心点检测时，当某个网格检测到物体时，它可能会认为这个物体的中心点在这个网格中，从而对物体重复检测，这个同一个物体就有多个目标框生成

YOLO算法在输出锚点框时，也会输出这个框的置信度，代表这个锚点框正确识别物体的概率。

非极大值抑制首先会找置信度最高的目标框(这里是0.9)，然后将IoU重叠较高的框去掉，然后找置信度第二高的框…以此重复。非极大值抑制的意思是只保留置信度最高的框，将其他置信度不是极大值的框去掉。

非极大值抑制的步骤是，通过神经网络对每个网格计算出一组参数(为了简化这里不加c1，c2，c3，只有pc和锚定框的中心点和长宽参数)

  1. 将概率pc≤0.6的框筛掉
  2. 取置信度最大的框，将和这个框的IoU较大的框筛掉
  3. 直到没有剩下需要筛的框为止

如果我们需要检测多个物体，则需要对每个物体检测都进行一次非极大值抑制。

3.8 Anchor Boxes

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

使用Anchor Boxes，我们能让单个网格一次性检测出多个物体。

当我们需要检测人和车时，假设人和车的中心点在一个网格内，我们就不能通过一个y来识别两个物体，最多只能识别其中的一个。

Anchor box做的是预选确定要框的比例(比如竖的框更贴合行人，横的框更贴合车辆)，然后对每种框都单独设置参数(比如右下角的y，就设置了2组16个参数，第一组参数识别行人，第二组参数识别车辆)。

使用anchor box之后，识别到的物体不仅会被分配到一个网格中，而且会分配给它一个和形状的重叠程度(IoU)最高的Anchor box。这样，如果你用两个Anchorbox(这里说明输出有16个维度)，就可以识别两个形状不同的物体了。

使用锚点框的算法有两种情况会处理不好

  1. 只使用了两个锚定框，但网格内有三个物体的中心点，即物体的数量多于锚定框的数量
  2. 识别出的两个物体属于相同的锚定框，即一个锚定框识别了多个物体

实际中这两种情况都较难发生。

使用anchor box可以让算法对不同形状的物体更有针对性的处理。一般人们会手工选定5到10个anchor box形状以涵盖要检测对象的形状。

还有一种方法能自动决定anchor box的形状，就是用k-平均算法将物体分成多个聚类，然后从每个聚类中选取一个最有代表性的anchor box形状。

4.1 什么是人脸识别

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

人脸验证：给一张图片和一个名字/ID，判断输入的图片是不是这个人，1对1问题

人脸识别比人脸验证要难很多，人脸识别是一个1对k的问题，你需要将输入的图片和数据库中的上百张图片进行匹配。如果用人脸验证的系统来做这件事，假设人脸验证的正确率是99%，那么人脸识别的正确率大概就是(99%)^500(假设数据库中有1000个人，则平均需要进行500次匹配)为0.65%，非常低。

因此想要在人脸识别中用上人脸验证的系统，需要人脸验证的正确率达到极高的水平

4.10 风格代价函数

发表于 2023-10-27 分类于 吴恩达深度学习4-卷积神经网络

和内容代价函数一样，我们选择神经网络中的某一层来决定风格代价。

“风格”被描述为层中不同通道间激活项的相关系数，即两个通道之间，不同的激活项组成不同的数字对，这些数字对用来计算相关系数，而相关系数用来描述风格。

对同一隐藏层的不同通道来说，什么是相关系数呢？举个例子说就是，如果一个通道凸显的特征是垂直边缘，而另一个通道凸显的特征是橙色，那么当垂直边缘出现时它大概率是橙色，则说明这两个通道是相关的，反之则说明是不相关的。

我们可以通过比较生成图片和原来风格图片的这两个通道之间相关系数的大小，来衡量两张图片的风格相似程度

过程是计算一个风格矩阵，矩阵大小为nc×nc，具体做法是选择隐藏层中的两个不同通道，让两个通道对应位置的aij相乘，最后将所有的aij累加起来构成一个矩阵值，这样的矩阵又称为gram matrix

我们将生成图片和风格图片同时计算矩阵G，则风格损失函数就是两个G之间的距离

作者在前面带了一个归一化常数，但无关紧要，因为风格函数最后是会乘以超参数β的。

你也可以使用多个隐藏层来计算风格代价函数，此时需要在前面乘一个超参数λ来平衡，这种方式能让风格即参考低级边缘特征，也参考高级特征
